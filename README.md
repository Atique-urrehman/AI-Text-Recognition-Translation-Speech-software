# AI Text Recognition, Translation & Speech App (React)

## Project Overview

This is a **React-based web application** that allows users to extract text from images, edit it, translate it into different languages, and listen to the translated output using text-to-speech.

The app focuses on **user experience and integration of intelligent APIs**, providing a smooth workflow from image input to translated audio output.

---

## Features

* 📸 Upload image to extract text (OCR integration)
* 🎥 Live camera input support
* ✏️ Edit extracted text before translation
* 🌐 Multi-language translation (user-selectable)
* 🔊 Text-to-speech for translated text
* 🎙️ Speech-to-text input (voice typing)
* 📝 Spell-check and word suggestions
* 🕘 History of translations
* 🌙 Dark mode & ☀️ Light mode UI
* laguages are:
  Urdu, Arabic, Spanish, japanies, english German, Portuguese, Chinese, Italian, Russian
---

## How It Works

1. User uploads an image or uses live camera
2. Text is extracted via OCR API
3. User can:

   * Edit text
   * Fix spelling (suggestions available)
   * Use voice input
4. Select target language
5. App translates text using API
6. Translated text can be played as audio

---

## Technologies Used

* ⚛️ React (Frontend)
* 🎨 HTML, CSS, JavaScript
* 🔌 OCR APIs (e.g., Tesseract OCR)
* 🌐 Translation APIs (e.g., Google Translate API)
* 🔊 Text-to-Speech APIs
* 🎙️ Web Speech API (Speech Recognition & Voice Input)
